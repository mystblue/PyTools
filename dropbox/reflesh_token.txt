🧭 ステップ1：Dropbox アプリの作成
- Dropbox App Console にアクセス
- 新しいアプリを作成し、OAuth2 設定で token_access_type=offline を有効にする
- App key と App secret を控えておく

🔐 ステップ2：Authorization Code の取得
以下の URL にアクセスして認証コードを取得します（<APP_KEY> を自分の値に置き換えてください）：
https://www.dropbox.com/oauth2/authorize?client_id=<APP_KEY>&response_type=code&token_access_type=offline


- 認証画面を進めると、authorization code が表示されます（1回限り有効）

🔄 ステップ3：リフレッシュトークンの取得
Postman または curl を使って、以下のように Dropbox API に POST します：
curl https://api.dropboxapi.com/oauth2/token \
  -d code=<AUTHORIZATION_CODE> \
  -d grant_type=authorization_code \
  -u <APP_KEY>:<APP_SECRET>


- 成功すると、access_token と refresh_token を含む JSON が返ってきます
